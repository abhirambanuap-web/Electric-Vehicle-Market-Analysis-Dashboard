⚡ Electric Vehicle Data Analysis Dashboard
📊 Project Overview

The Electric Vehicle Data Analysis Dashboard is an Excel-based data analytics project designed to explore trends in electric vehicle adoption, manufacturer performance, electric range, and regional distribution. The project converts raw EV registration data into meaningful insights using interactive dashboards, KPIs, and visualizations.

The goal of this project is to demonstrate data analysis, data visualization, and dashboard development skills using Microsoft Excel. By leveraging pivot tables, charts, slicers, and KPI indicators, the dashboard allows users to analyze electric vehicle trends efficiently.

Electric vehicles are becoming increasingly important in the transition toward sustainable transportation. This dashboard provides a clear view of how EV adoption is growing, which manufacturers dominate the market, and how electric vehicles are distributed geographically.

📁 Dataset

The dataset contains 114,595 electric vehicles and includes key attributes such as:

Vehicle type (BEV / PHEV)

Manufacturer and model

Model year

Electric range

State distribution

CAFV (Clean Alternative Fuel Vehicle) eligibility

This data enables analysis of EV market growth and technological development.

📌 Key Performance Indicators (KPIs)

The dashboard highlights several important metrics:

Total Vehicles: 114,595

Battery Electric Vehicles (BEV): 87,767 (76.59%)

Plug-in Hybrid Electric Vehicles (PHEV): 26,828 (23.41%)

These KPIs provide a quick overview of the EV market composition.

📈 Dashboard Insights
🚗 Top EV Manufacturers

The dashboard identifies the Top 10 manufacturers by number of vehicles.
Tesla dominates the market with over 52,000 vehicles, followed by Nissan and Chevrolet.

🔋 CAFV Eligibility

Vehicles are categorized based on Clean Alternative Fuel Vehicle eligibility:

Eligible for incentives

Eligibility unknown

Not eligible due to battery range

More than 50% of vehicles qualify for clean fuel incentives, indicating strong policy support for EV adoption.

📅 EV Registrations by Model Year

The model year analysis shows rapid growth in EV registrations in recent years, reflecting increasing demand and technological advancement in the electric vehicle industry.

⚡ Electric Range Comparison

A treemap visualization highlights Top EV brands with the highest electric range, helping compare battery performance across manufacturers.

🗺️ EV Distribution by State

The map visualization displays electric vehicle distribution across states, providing insights into regional adoption patterns.
